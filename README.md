# Hi everyone 👋

I am a promising creative individual currently living in 📍Villingen, Germany. I enjoy designing and programming things. I create websites both from a design perspective — focusing on usability and aesthetics — and from a technical one, aiming to implement the required features as efficiently as possible using programming languages and modern technologies.

My journey into the world of coding began a couple of years ago, when I first touched HTML during one of the boring IT lessons at school. Today I work with programming almost every day, as I am doing an apprenticeship as a software developer at a major automotive corporation called Aumovio. I have the privilege of working with excellent software engineers and learning different aspects of programming from their experience.

My passion for web design awakened relatively recently — about half a year ago. I wanted my websites not only to function well but also to look aestetic and be user-friendly, which is when Figma cam into play for me.
